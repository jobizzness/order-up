# Order Up — Restaurant Order Tracking System

**Order Up** is a real-time restaurant order tracking application that streamlines kitchen-to-table communication. Built for restaurants, cafés, and food service businesses that need fast, reliable order management.

## Why Order Up?

Managing restaurant orders across front-of-house, kitchen, and delivery is chaotic without the right tools. Order Up gives your team a single source of truth — from the moment an order is placed to the moment it's served.

- **Real-time order tracking** — see every order's status as it moves through your kitchen
- **Admin dashboard** — manage menus, monitor orders, and track performance from one place
- **Mobile-first customer app** — guests can browse the menu, place orders, and track progress from their phone
- **Kitchen display integration** — kitchen staff see incoming orders instantly, no paper tickets needed
- **Multi-location ready** — scale from a single restaurant to a chain

## Who Is This For?

- Restaurant owners looking for a **digital order management system**
- Cafés and fast-casual spots that need **real-time kitchen-to-customer updates**
- Food trucks and pop-ups that want **mobile ordering and tracking**
- Franchise operators needing **centralized order monitoring across locations**

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install Dependencies

```bash
npm run install:all
```

### Run the Admin Dashboard

```bash
npm run dev:admin
```

### Run the Mobile App

```bash
npm run dev:app
```

### Build for Production

```bash
npm run build:admin
npm run build:app
```
