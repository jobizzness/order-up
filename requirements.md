# Restaurant Table Booking App — Requirements

## Overview

Online Restaurant Reservation System with three panels:
- **User Panel** (iOS + Android) — for customers to book tables, view menus, and manage reservations
- **Restaurant Manager Panel** (Web) — for restaurant staff to manage bookings, tables, and operations
- **Admin Dashboard** (Web) — for platform admins to manage restaurants, users, and analytics

---

## User Panel (Mobile App)

### Core Features

- [ ] **User Registration / Login** — email, phone, social auth
- [ ] **Location-Based Search** — find nearby restaurants
- [ ] **Restaurant Discovery** — browse by cuisine, rating, price, availability
- [ ] **Real-Time Table Availability** — view available slots
- [ ] **Online Table Booking** — select date, time, party size
- [ ] **Waitlist Management** — join waitlist when fully booked
- [ ] **Special Requests** — dietary restrictions, seating preferences (window, outdoor, etc.)
- [ ] **QR Menu** — contactless digital menu scanning
- [ ] **In-App Payments** — pay for reservations, pre-orders, deposits
- [ ] **Booking History** — past and upcoming reservations
- [ ] **Push Notifications** — booking confirmations, reminders, offers
- [ ] **Loyalty & Rewards** — points, cashback, exclusive offers
- [ ] **Ratings & Reviews** — rate dining experience

---

## Restaurant Manager Panel (Web)

### Core Features

- [ ] **Dashboard** — overview of today's bookings, occupancy, revenue
- [ ] **Reservation Management** — view, confirm, modify, cancel bookings
- [ ] **Table Management** — configure table layout, capacity, sections
- [ ] **Waitlist Management** — manage waitlisted guests
- [ ] **QR Menu Builder** — create and update digital menus
- [ ] **Booking Rules** — set hours, availability, blackout dates
- [ ] **Staff Management** — add staff, assign roles
- [ ] **Offers & Promotions** — create discounts, special events
- [ ] **Customer Insights** — guest profiles, preferences, visit history
- [ ] **Notifications** — SMS/email to guests

---

## Admin Dashboard (Web)

### Core Features

- [ ] **Platform Overview** — total restaurants, bookings, revenue
- [ ] **Restaurant Management** — onboard, approve, suspend restaurants
- [ ] **User Management** — manage customer accounts
- [ ] **Analytics & Reporting** — booking trends, revenue reports
- [ ] **Commission & Payments** — track payouts to restaurants
- [ ] **Content Management** — featured restaurants, promotions
- [ ] **Support Tickets** — handle disputes and issues
