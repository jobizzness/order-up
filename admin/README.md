# Order Up — Admin Dashboard

Restaurant order tracking admin panel built with Next.js, React, and Tailwind CSS.

## Modules

### Auth
- Staff login and session management
- Role-based access control (owner, manager, kitchen staff)

### Logistics
- Kitchen display and order routing
- Channel management (dine-in, takeout, delivery)
- Real-time status updates

### Orders
- Create, view, and manage incoming orders
- Order queue and kanban board views
- Order history and search

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Language:** TypeScript
